# Project previews

Place project preview images or GIFs here.

**Reference in data:** Use paths like `/projects/your-file.jpg` or `/projects/your-file.gif` in `app/data/projects.ts`.

**Recommended:**
- **Format:** JPG or PNG for static screenshots; GIF for short animated previews.
- **Aspect ratio:** 16:9 (e.g. 1200×675 or 1600×900) so they fit the browser mockup.
- **Dimensions:** At least 1200px wide for sharp display on retina screens.
